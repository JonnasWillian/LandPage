O que é uma LandPage ?


Uma landing page é uma página específica de conversão. Ela possui vários elementos voltados para tornar o visitante em lead, como um copy atrativo, formulário, CTA, entre outros. São páginas muito utilizadas nas campanhas de marketing digital e costumam ter boas taxas de conversão.

Em outras palavras é uma página feita para captar a atenção de usuários de público específico (ou não), para encaminhar eles para algum outro tipo de página principal




Qual foi o intuito dessa LandPage


Está LandPage é apenas uma página de teste, sem fins lucrativos ou nada em específico, apenas criada para treinar habilidades em HTML e CSS e por um projeto simples e bem feito, com variação de colorações leves e efeitos chamativos para testes e amostra de habilidades
